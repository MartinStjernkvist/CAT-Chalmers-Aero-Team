# CAT mission tools

This package is meant to store some simple mission tools to write scenarios operating a drone.

### Dependencies

- Ensure you have python >=3.12 installed.

```shell
redis-server
```

### Installation

- Clone the repo
- Activate your python environment (in case you are using venv or conda)
- Install the package (as editable)

```shell
pip install -e .
```
